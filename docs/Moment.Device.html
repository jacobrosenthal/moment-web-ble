Device

Moment. Device

new Device()

Source:
  • moment-web-ble.js, line 123

Represents a Moment device connection from the browser, and provides methods for connecting and executing JavaScript code on Moment using the features provided by the Moment SDK.

Example
// connect to device
var device = new Moment.Device();
device.connect();
device.run("5 + 5;"); // execute JavaScript code on Moment
device.disconnect();
